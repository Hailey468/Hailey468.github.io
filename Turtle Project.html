Refactoring in Python Turtle


About the Project

In this project, Python's Turtle is used to create a graphical scene. For this I used the building blocks draw_rectangle, draw_triangle, draw_circle, and draw_oval to create functions that draw composite shapes such as a barn, fence, cow, and more. Compared to drawing each element with the basic shapes, the refactored version made it much easier to draw repeated element with just altering their size. To create my refactored code I made the elements' size in the previous unfactored code the base size for the refactored version. This allows for the elements size to be changed to create duplicates by reducing or enlarging the element by a specified factor, in this code that factor is p. When calling functions reapeatedly, such as draw_tree or draw_cow, I used coords so that I would not have to set new coordinates for each repeated element and instead just change the sizing.

Sample Code

This function uses ovals, circles, rectangles in order to draw a cow. The coords are set so that a cow will draw at the specified coordinates without needing to repeat the function for each location. The variable p is the proportion of the cow's original size the duplicate will take. For a p value of larger than one the cow will be larger than the original by that percent. The opposite is true for a value of less than one, which will cause the cow to be smaller. In this code, a value of .7 for p will create a cow 70% the size of the original.


turtle: Calls the graphics object for drawing

start_x: The x value that the function shall begin drawing at

start_y: The y value that the function shall begin drawing at

p: the percentage of the original size the object will take in decimal form

coords = [ (100,-170,1), (195,-200,.7), (310,-210,.6), ] for coord in coords: start_x,start_y,p = coord draw_cow(turtle,start_x,start_y,p)

Final Scene

This is the completed turtle scene that features a barn, fence, multiple trees, three cows, and a sun.



This scene uses multiple composite shapes created from basic turtle functions.The barn is made with a large red rectangle, a black triangle, black lines, and 4 long angled rectangles. The trees consist of brown rectangles and a green circle, all varying sizes. The fences are made using both horizontal an vertical rectangles. The cows are created with multiple sizes of ovals, circles, and a rectangle for the tail. The sun is a circle, and the ground a rectangle.
